# 散策スポット一覧

## 1. 淡路夢舞台 ⭐確定

- **優先度:** 最優先（今回の軸）
- **営業時間:** 7:00〜22:00（無料エリア）／あわじグリーン館10:00〜18:00
- **所要時間:** 45〜75分
- **おすすめルート:** 地下駐車場 → 水庭 → 円形フォーラム → 海回廊 → 百段苑（少し） → 展望テラス
- **雨対策:** 雨が強い時は屋外回廊を短縮して、あわじグリーン館中心に
- [公式サイト](https://www.yumebutai.co.jp/)

## 2. あわじグリーン館 ⭐雨対策

- **優先度:** 雨の日の夢舞台内バックアップとして最優先
- **営業時間:** 10:00〜18:00（最終入館17:30）
- **企画展:** 5/16〜7/12『花の楽園〜植物のある暮らし〜』
- [公式サイト](https://awaji-botanicalgarden.com/)
