# 散策スポット一覧

## 1. 淡路夢舞台 ⭐確定

- **優先度:** 最優先（今回の軸）
- **営業時間:** 百段苑・展望テラス・海回廊・山回廊 7:00〜22:00（無料）／あわじグリーン館 10:00〜18:00（最終入館17:30）
- **所要時間:** 45〜75分
- **おすすめルート:** 地下駐車場 → 水庭 → 円形フォーラム → 海回廊 → 百段苑（少し） → 展望テラス → 駐車場へ戻る
- **雨対策:** 百段苑は短めにして、回廊・水庭・あわじグリーン館寄りで回る
- [Googleマップで開く](https://www.google.com/maps/search/?api=1&query=%E6%B7%A1%E8%B7%AF%E5%A4%A2%E8%88%9E%E5%8F%B0)

## 2. 多賀の浜

- **優先度:** 雨が弱ければ行く
- **おすすめ時間:** 17:20〜18:10のどこか、15〜30分だけ
- **駐車場:** 250台・500円
- **特徴:** 播磨灘に沈む夕日を眺められる海岸
- **注意:** 雨が強ければ無理しない。夕日を最後まで粘ると返却がタイトに
- [Googleマップで開く](https://www.google.com/maps/search/?api=1&query=%E5%A4%9A%E8%B3%80%E3%81%AE%E6%B5%9C%E6%B5%B7%E6%B0%B4%E6%B5%B4%E5%A0%B4)

## 3. 絵島・岩屋港周辺

- **優先度:** 浜ちどり前の短時間候補
- **使い方:** 浜ちどりに少し早く着きそうな時、10〜15分だけ寄る
- **雨対策:** 車から雰囲気を見る程度でもOK
- [Googleマップで開く](https://www.google.com/maps/search/?api=1&query=%E7%B5%B5%E5%B3%B6+%E5%B2%A9%E5%B1%8B%E6%B8%AF)

## 4. 淡路SA・大観覧車

- **優先度:** 帰り道の時間調整候補
- **使い方:** 雨で多賀の浜をやめた時、または夕食が早く終わって少しだけ余裕がある時
- **注意:** 帰りの工事・渋滞があるので、無理に入れない
- [Googleマップで開く](https://www.google.com/maps/search/?api=1&query=%E6%B7%A1%E8%B7%AFSA+%E5%A4%A7%E8%A6%B3%E8%A6%A7%E8%BB%8A)
